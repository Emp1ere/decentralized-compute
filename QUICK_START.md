# Быстрый старт: проверка работы системы

## Запуск системы

1. **Запустите Docker Compose:**
   ```bash
   docker-compose up
   ```

2. **Откройте веб-интерфейс:**
   - Узел 1: http://localhost:5000
   - Узел 2: http://localhost:5001

## Проверка работы в интерфейсе

### Шаг 1: Регистрация
1. Откройте вкладку **"Обзор"**
2. Нажмите **"Зарегистрироваться"**
3. Скопируйте полученный `api_key`

### Шаг 2: Получение задачи
1. Перейдите на вкладку **"Работа"**
2. Вставьте `api_key` в поле (или он уже будет там из вкладки "Обзор")
3. Нажмите **"Получить задачу"**
4. Вы увидите задачу с описанием:
   - Название задачи
   - Категория (например, "Астрофизика")
   - Описание (что моделируется)
   - Contract ID
   - Work units required
   - Difficulty
   - Тип вычислений

### Шаг 3: Выполнение задачи в браузере
1. Нажмите **"Выполнить в браузере"**
2. Для **простых задач (simple_pow)**: выполняется полный PoW
3. Для **астрофизических задач**: выполняется упрощённая версия (до 1000 единиц работы для скорости)
4. Следите за прогрессом в сообщении
5. После завершения результат автоматически отправится на сервер

### Шаг 4: Проверка результата
1. После отправки вы увидите сообщение об успехе или ошибке
2. Перейдите на вкладку **"Обзор"** и нажмите **"Показать баланс"** для проверки награды

## Просмотр контрактов

1. Перейдите на вкладку **"Контракты"**
2. Вы увидите таблицу со всеми доступными задачами:
   - Название
   - Категория
   - Описание
   - Единиц работы
   - Сложность
   - Награда

## Типы задач

### Тестовые задачи:
- **sc-001**: Простая хеш-задача (1000 единиц работы, награда 10)
- **sc-002**: Сложная хеш-задача (5000 единиц работы, награда 50)

### Астрофизические задачи:
- **astro-001**: Космологическая симуляция (100,000 единиц работы, награда 500)
- **astro-002**: Моделирование сверхновой (80,000 единиц работы, награда 400)
- **astro-003**: МГД джетов и аккреции (60,000 единиц работы, награда 300)
- **astro-004**: Радиационный перенос (70,000 единиц работы, награда 350)
- **astro-005**: Гравитационные волны (150,000 единиц работы, награда 750)

## Примечания

- **В браузере**: Астрофизические задачи выполняются в упрощённом виде (до 1000 единиц работы) для скорости
- **Воркеры**: Выполняют полные вычисления для всех типов задач
- **Верификация**: Оркестратор проверяет результаты через пересчёт с тем же seed

## Проверка работы воркеров

Воркеры автоматически:
1. Регистрируются на оркестраторе
2. Получают задачи
3. Выполняют вычисления (полный объём работы)
4. Отправляют результаты
5. Проверяют баланс

Логи воркеров можно посмотреть через:
```bash
docker logs client_worker_1
docker logs client_worker_2
```

## Мониторинг

- **Вкладка "Статистика"**: Метрики узла (длина цепочки, клиенты, pending, запросы, ошибки)
- **Вкладка "Блокчейн"**: Просмотр всех блоков
- **Вкладка "Задачи"**: История сданных работ
