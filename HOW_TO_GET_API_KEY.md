# Как посмотреть API-ключ вычислителя

Этот файл — короткая инструкция, где взять `api_key` для `desktop_agent`.

## Вариант 1 (рекомендуется): через веб-логин

1. Откройте веб-интерфейс: `http://localhost:8080/dashboard`
2. Перейдите на вкладку `Обзор`.
3. Введите логин и пароль.
4. Нажмите `Войти`.

После входа фронтенд получает `api_key` от `/auth/login` и сохраняет его в сессии.

## Вариант 2: из localStorage браузера

Если вы уже вошли в вебе, откройте DevTools -> Console и выполните:

```js
localStorage.getItem('dc_auth_api_key')
```

Будет выведен текущий `api_key`.

## Вариант 3: напрямую через API

Отправьте `POST` на `http://localhost:8080/auth/login` с JSON:

```json
{
  "login": "your_login",
  "password": "your_password"
}
```

В ответе будет поле `api_key`.

## Куда вставлять ключ в Desktop Agent

1. Запустите `desktop_agent/start_desktop_agent.bat`.
2. В поле `API ключ вычислителя` вставьте значение `api_key`.
3. Нажмите `Проверить /me` для проверки доступа.

## Важно по безопасности

- Не отправляйте `api_key` в чаты и скриншоты.
- Не публикуйте ключ в репозитории.
- Если ключ скомпрометирован — создайте новый аккаунт/ключ и замените его в агенте.
